# PopX Assignment

React JS implementation of the provided Adobe XD design.

## Features

* Welcome Screen
* Login Screen
* Registration Screen
* Account Settings Screen
* React Router Navigation
* Form Validation
* Local Storage Based Authentication
* Responsive Mobile-Centered UI

## Tech Stack

* React JS
* Vite
* React Router DOM
* CSS

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
